Calcolo Vettoriale

▶

Notazioni

Componenti: a⃗ = (ax, ay, az)
Complesso: a⃗ = ax + j·ay
Polare: a⃗ = |a| ∠ θ

Modulo e direzione

|a⃗| = √(ax² + ay²)
θ = atan2(ay, ax)

Conversione polare ↔ componenti

ax = |a|·cosθ     ay = |a|·sinθ

Somma

a⃗ + b⃗ = (ax+bx,  ay+by)

Prodotto scalare

a⃗ · b⃗ = axbx + ayby = |a||b|cosθ
Risultato: scalare. Se = 0 ⇒ a ⊥ b

Prodotto vettoriale (nel piano)

a⃗ × b⃗ = (axby − aybx) k̂
|a⃗ × b⃗| = |a||b|sinθ
Risultato: vettore ⊥ al piano. NON commutativo: a×b = −(b×a)

Cinematica del Punto

▶

Coordinate cartesiane

P⃗(t) = x(t) î + y(t) ĵ
v⃗ = ẋ î + ẏ ĵ
a⃗ = ẍ î + ÿ ĵ

Coordinate polari

P⃗ = ρ · êρ

v⃗ = ρ̇ êρ + ρθ̇ êθ

a⃗ = (ρ̈ − ρθ̇²) êρ + (ρθ̈ + 2ρ̇θ̇) êθ

Significato dei termini dell'accelerazione

ρ̈
Radiale propria
−ρθ̇²
Centripeta (verso O)
ρθ̈
Angolare
2ρ̇θ̇
Coriolis

Derivata dei versori polari

dêρ/dt = θ̇ · êθ
dêθ/dt = −θ̇ · êρ

Moto angolare uniformemente accelerato

ω(t) = ω0 + ω̇·t
θ(t) = θ0 + ω0t + ½ω̇t²

Cinematica del Corpo Rigido

▶

Gradi di liberta

GdL = 3 × Ncorpi − ∑(vincoli rimossi)

Cerniera terra: −2  |  Cerniera int.: −2  |  Carrello: −1
Incastro: −3  |  Manicotto: −2  |  Pattino: −2

Teorema di Rivals

v⃗B = v⃗A + ω⃗ × (B − A)

a⃗B = a⃗A + ω̇⃗ × (B−A) + ω⃗ × (ω⃗ × (B−A))

ω × (B−A): ⊥ all'asta, modulo ω|BA|

ω̇ × (B−A): tangenziale, ⊥ all'asta, modulo ω̇|BA|

ω × (ω × (B−A)): centripeta, da B verso A, modulo ω²|BA|

Prodotto vettoriale nel piano

ω⃗ × r⃗ = ω k̂ × (rx î + ry ĵ) = (−ω ry,  ω rx)
Ruota r di 90° antiorario e moltiplica per ω

Puro rotolamento

vcontatto = 0
vcentro = ω × R   (modulo ωR, orizzontale)
vcima = 2ωR
acentro = ω̇ R   (orizzontale)

Moti relativi

v⃗ass = v⃗rel + v⃗tras

a⃗ass = a⃗rel + a⃗tras + 2ω⃗ × v⃗rel

Il termine 2ω × vrel e l'accelerazione di Coriolis.

Meccanismi

▶

Chiusura cinematica

Loop: OA⃗ + AB⃗ + BO⃗ = 0⃗

Ogni asta: L cos(α) î + L sin(α) ĵ

Derivata per velocita

d/dt [L cos(α)] = −L sin(α) · α̇
d/dt [L sin(α)] = L cos(α) · α̇
Le lunghezze L sono costanti!

Derivata per accelerazione

d²/dt² [L cos(α)] = −L cos(α)·α̇² − L sin(α)·α̈
d²/dt² [L sin(α)] = −L sin(α)·α̇² + L cos(α)·α̈

Glifo oscillante

v⃗ass = v⃗rel (lungo guida) + v⃗tras (ω × r, ⊥ raggio)

a⃗ass = a⃗rel + a⃗tras + 2ω⃗ × v⃗rel

Statica — Reazioni Vincolari

▶

Equilibrio

∑Fx = 0     ∑Fy = 0     ∑Mpolo = 0

3 equazioni scalari per corpo nel piano.

Reazioni vincolari

Cerniera a terra: Hx, Hy  (2 incognite)
Carrello: N ⊥ piano  (1 incognita)
Incastro: Hx, Hy, M  (3 incognite)
Manicotto: N, M  (2 incognite)
Pendolo/biella: F lungo l'asta  (1 incognita)

Isostaticita

N° incognite = N° equazioni ⇒ isostatico (risolvibile)
N° incognite > N° equazioni ⇒ iperstatico
N° incognite < N° equazioni ⇒ labile

Molla

Fmolla = k · (l − l0)
l = lunghezza attuale, l0 = lunghezza indeformata

Azioni Interne

▶

Definizioni

N = Sforzo Normale (trazione +, compressione −)
T = Taglio
M = Momento flettente

Relazioni differenziali

dT/ds = q   (carico distribuito)
dM/ds = T   (il momento e l'integrale del taglio)

Regole per i diagrammi

Tra forze concentrate: T costante, M lineare
Forza concentrata: salto in T
Coppia concentrata: salto in M
Dove T = 0: M ha max o min
Bielle (2 cerniere, no carichi): solo N, T=0, M=0

Dinamica — Equazioni Cardinali

▶

Prima equazione cardinale (forze)

∑F⃗ext = m · a⃗G

Seconda equazione cardinale (momenti)

Polo in G (baricentro): ∑MG = JG · ω̇

Polo in O (punto fisso): ∑MO = JO · ω̇

Teorema assi paralleli (Huygens-Steiner)

JO = JG + m · d²   (d = distanza G−O)

Energia cinetica

T = ½ m vG² + ½ JG ω²

Se ruota attorno a punto fisso O: T = ½ JO ω²

Bilancio delle Potenze

▶

Principio

∑Pesterne = dT/dt

Potenze

Forza: P = F⃗ · v⃗ = |F|·|v|·cosθ
Coppia: P = C · ω
Peso: P = −m g vy

Derivata energia cinetica

dT/dt = m · a⃗G · v⃗G + JG · ω̇ · ω

Potenze nulle (non compaiono!)

Cerniera a terra: v = 0 ⇒ P = 0
Carrello: F ⊥ v ⇒ P = 0
Cerniera interna: azione-reazione si cancellano

Attrito e Rotolamento

▶

Attrito

Statico: |Fattr| ≤ fs · N   (disequazione!)

Dinamico: Fattr = fd · N   (equazione, opposta a v)

fs > fd  sempre

Verifica aderenza

1. Ipotesi: puro rotolamento (Ft incognita)
2. Calcola Ft e N
3. Verifica: |Ft| ≤ fs · N ?
Si ⇒ OK, non slitta
No ⇒ slitta, usa fd

Resistenza al rotolamento

Momento resistente: Mv = fv · N
Forza equivalente: Fv = fv · N / R
Si oppone alla rotazione

Trasmissioni

▶

Rapporto di trasmissione

τ = ωuscita / ωingresso
Pulegge con fune: τ = Rin / Rout
Ingranaggi: τ = zin / zout

Rendimento

Moto diretto (carico resiste):
Pmotore · ηD = Putile  ⇒  dividi per η

Moto retrogrado (carico aiuta):
Pcarico · ηR = Pmotore  ⇒  moltiplica per η

Diretto = "se spengo il motore, il sistema si ferma". Retrogrado = "se spengo il motore, il sistema si muove da solo".

Inerzia equivalente ridotta al motore

Jeq = Jm + τ² · Jcarico
Per masse traslanti: Jeq,m = m · (v / ωm)² = m · (τR)²

Bilancio completo con trasmissione

Cm·ωm = Jeq·ω̇m·ωm + Pres / ηD

A regime (ω̇ = 0):
Cm·ωm = Pres / ηD

Curva caratteristica del motore

Cm = Cm0 + S · (ωm − ωs)

Cm0 = coppia allo spunto (ω=0)
ωs = velocita di sincronismo
S = pendenza (negativa)
